Gemini API 키 발급 (무료)

API 키가 필요한 이유

Error Auto-Fixer는 Google Gemini AI를 사용하여 에러를 분석합니다. API 키는 무료로 발급받을 수 있으며, 신용카드 등록이 필요 없습니다.

✓
완전 무료! Gemini API 무료 티어로 일일 최대 250회 분석이 가능합니다. 별도 과금이 없습니다.

사용 방법

전체 사용 흐름

에러 발생

프로젝트에서 컴파일 에러나 런타임 에러가 발생하면 자동으로 캡처됩니다.

Error Auto Fixer 창 열기

Tools > Error Auto Fixer > Open Window

에러 선택 & 분석

좌측 목록에서 에러를 선택하고 [분석] 버튼을 클릭합니다.

AI 분석 결과 확인

Gemini AI가 에러 원인을 분석하고 진단 결과를 표시합니다 (1~5초).

diff 미리보기를 확인하고 [수정 적용] 버튼으로 원클릭 패치합니다.

에러 분석 과정

  1. 에러 목록에서 분석하려는 에러를 클릭하여 선택합니다.
  2. 좌측 하단의 [분석] 버튼을 클릭합니다.
  3. "Gemini API로 분석 중..." 메시지가 표시되며, 보통 1~5초 내에 결과가 나옵니다.
  4. 동일한 에러를 다시 분석하면 캐시된 결과가 즉시 표시됩니다.
i
콘솔 동기화: [스캔] 버튼을 누르면 Unity 콘솔과 에러 목록이 동기화됩니다. 이미 해결된 컴파일 에러는 자동으로 제거됩니다.

코드 수정 적용

  1. 분석 결과에 수정 가능이 표시되면 diff 미리보기를 확인합니다.
  2. 수정 내용이 적절하다고 판단되면 [수정 적용] 버튼을 클릭합니다.
  3. 확인 대화상자에서 [적용]을 선택합니다.
  4. 파일이 자동으로 수정되고, Unity가 리컴파일을 시작합니다.
!
수정이 마음에 들지 않으면?
Git을 사용 중이라면 git checkout -- 파일경로 또는 IDE의 Discard Changes로 간편하게 되돌릴 수 있습니다.

자주 묻는 질문 (FAQ)

API 키를 입력했는데 "연결 테스트 실패"가 뜹니다
  • API 키를 복사할 때 앞뒤 공백이 포함되지 않았는지 확인하세요.
  • Google AI Studio에서 키가 활성화 상태인지 확인하세요.
  • 네트워크 연결 (방화벽, 프록시 등)을 확인하세요.
  • VPN을 사용 중이라면 비활성화 후 재시도하세요.
비용이 정말 발생하지 않나요?

네, Google Gemini API 무료 티어를 사용합니다. 카드 등록 없이 일일 최대 250회 분석이 가능하며, 기간 제한 없이 영구 무료입니다.

"Rate Limit 초과" 에러가 발생합니다

무료 티어는 분당 10회 요청 제한이 있습니다. 잠시 기다렸다가 다시 시도하세요. 설정에서 gemini-2.5-flash-lite (분당 15회, 일일 1,000회)로 모델을 변경하면 여유가 생깁니다.

AI가 제안한 수정이 틀린 것 같습니다

LLM 특성상 100% 정확한 수정을 보장하지 않습니다. 반드시 diff 미리보기를 확인한 후 적용하세요. 잘못된 수정을 적용했다면 Git의 discard changes로 되돌릴 수 있습니다.

에러 목록에 표시되지 않는 에러가 있습니다
  • [스캔] 버튼을 눌러 Unity 콘솔과 동기화하세요.
  • 설정에서 "자동 에러 캡처"가 켜져 있는지 확인하세요.
  • Warning 레벨 로그는 캡처되지 않습니다 (Error, Exception, Assert만 캡처).
내 소스코드가 외부로 전송되나요?

에러 분석을 위해 에러 관련 파일의 코드만 Google Gemini API로 전송됩니다. 전체 프로젝트가 전송되지 않으며, 전송 범위는 에러가 발생한 파일/메서드로 최소화됩니다.

패키지를 제거하려면 어떻게 하나요?

Window > Package Manager에서 "Error Auto Fixer"를 선택한 뒤 Remove 버튼을 클릭하면 됩니다.
